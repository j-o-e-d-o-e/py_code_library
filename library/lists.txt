data structures

- mutable collections
- pop() pops the last item off the stack and returns its value
- remove(x) removes the first occurrence of 'x'
- del() deletes element at a specific index
- list(s) converts string to list
- join() turns a list of characters into a string

a = [2,1,0,3]
del(a[2])
a #prompts: [2, 1, 3]

l = ["a","b","c"]
"".join(l)
